Project Overview

A content-based movie recommender system that suggests movies to users based on the content of the movie that they have searched for. The system utilizes data from the 15,000 most popular movies on TMDB (The Movie Database) and make use of the cosine similarity algorithm to calculate movie similarity.

Features

  • User registration and authentication.
  • Movie search and recommendations.
  • Display detailed movie information using TMDB API.
  • User-friendly interface with HTML, CSS, and JavaScript.
  • Details and recommendations based on recommendations.

Technologies Used

  • Python (Flask) for the backend.
  • Cosine similarity algorithm for content-based recommendations.
  • TMDB API for fetching movie details.
  • HTML&CSS for user interface design.
  • Jinja2 Templating for dynamic elements and interactions.
